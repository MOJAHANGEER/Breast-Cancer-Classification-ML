# 🧠 Breast Cancer Classification using Neural Network

## 📌 Project Overview

This project focuses on building a **Breast Cancer Classification System** using **Machine Learning and Deep Learning techniques**. The goal is to accurately predict whether a tumor is **malignant (cancerous)** or **benign (non-cancerous)** based on medical features.

A **Neural Network model** is implemented to achieve high accuracy and reliability, making this project a strong demonstration of applied data science and deep learning skills.

## 📊 Dataset

The dataset contains medical diagnostic measurements used to classify tumors.

Typical features include:

* Radius
* Texture
* Perimeter
* Area
* Smoothness

Target:

* **0 → Malignant**
* **1 → Benign**
